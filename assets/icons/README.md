# Soplos Welcome Icon Placeholder

This directory will contain the application icons.

For now, the application will use fallback system icons.

To add the proper icon:
1. Create/copy com.soplos.welcome.png (128x128 recommended)
2. Optionally create additional sizes in subdirectories:
   - 16x16/
   - 24x24/ 
   - 32x32/
   - 48x48/
   - 64x64/
   - 128x128/
   - 256x256/

The icon should represent the Soplos Linux welcome experience.
